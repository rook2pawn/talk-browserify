Browserify

What is it? A way to use requires In your client side code

Install

npm install -g browserify

We can now

  • use thousands upon thousands of free available modules in npm in our client
  • use a large number of core node modules like querystring in our client
  • use Our own files and simply require them in! in our client

Lets use a free open source module we find on npmjs

  • For example,
  • async
  • and many many more
  • most usable in your client

How we do it

Some HTML

<script src='bundle.js' type='text/javascript'></script>

Some Javascript index.js 

              var async = require('async')
              async.series([
                function(done) {
                },
                function(done) {
                }
              ])
we turn index.js INTO bundle.js this way

browserify index.js -o bundle.js

Using our own 'module' like an npm module

  • We can use our own code and require it
  • Just like a published module
  • To make your code more organized with small files of exact functionality

Example for our own code

Say we want to write our own library, lib.js 

              exports.myfunction = function(text) {
                return text.toUpperCase()
              }

Now we can include lib.js


              var mylib = require('./lib.js')
              console.log(mylib.myfunction("cyberwizard")) 
              // outputs -> "CYBERWIZARD"
